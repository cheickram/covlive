Nombre DE Décès

Description

Le graphique ci-dessous montre de façon comparative la vitesse d'evolution du nombre de décès dans le monde entier.

Analyse

✱ Le premier graphique nous donne une idée globale de l’évolution du nombre de décès dans le monde entier au fil du temps, soit un total de 315.000 environ.
NB: ce graphique qui montre simplement l'évolution des décès confirmés au fil du temps n’est pas très utile pour répondre à la question de savoir comment la vitesse de l'épidémie se compare entre les différents pays. En effet, l'épidémie de COVID-19 n'a pas commencé en même temps dans tous les pays.

✱ Le deuxième graphique est conçu pour permettre comparaison entre chaque pays. cette comparaison commence le jour où ce pays a eu au moins 5 décès confirmés.
Cela nous permet de comparer la vitesse à laquelle le nombre de décès confirmés a augmenté après que l'épidémie a atteint un stade similaire dans chaque pays. Choisissons par exemple l’Amérique du Nord, nous remarquons que les États-Unis est le pays dont la trajectoire est plus élevée après avoir enregistré 5 décès comparé à d’autres pays comme le Canada ou le Mexique. En raison de la grande variation dans les données dans le nombre de morts, une échelle logarithmique a été utilisée.

✱ Le troisième graphique présente le nombre de décès par jour et par pays. Exemple: Choisissons le continent Europe, et mettons pause sur le graphique le 3 Avril 2020, Nous observons que l’Espagne a le plus grand nombre de décès (950 décès), ensuite vient l’Italie avec 760 décès.

✱ Le quatrième graphique est une table de fréquence qui présente le taux de mortalité calculé en faisant le rapport entre le nombre de décès et le nombre d’habitants.
Nous comparons ici le nombre de décès et le taux de mortalité. Nous voulons savoir s’il faut ou pas regarder uniquement le nombre de décès ou le taux de mortalité pour conclure que le virus a beaucoup affecté un pays par rapport à un autre. Si nous prenons par exemple le cas de United Kingdom qui a enregistré le plus grand nombre de décès soit 32692 pour 66 millions d’habitants environ, qui ne représente que 0.05% de taux de mortalité. Pourtant si nous adoptons une vue globale pour les pays, la table présente que le pays avec le plus grand taux de mortalité est San Marino (41 morts sur 33000 habitants) avec 0.12%. Donc on conclut qu’il faut mettre l’accent sur le taux de mortalité afin de voir réellement l’impact du virus sur la population de chaque pays.
