Nombres de Tests

Description

Denombremenet du nombre de tests effectués par jour par milliers de personnes.

Analyse

✱ Le premier graphique montre combien de tests sont effectués par jour par milliers de personnes. Considérons donc le top 10, le pays qui fait le plus de tests par milles personnes en Europe c’est l’Island, puis vient le Luxembourg puis la Lituanie etc.

✱ Le deuxième graphique montre le nombre total de tests effectués. De façon général on remarque qu’en Amérique ce sont les États-Unis qui font plus de tests par jour.

✱ Le troisième graphique reprend le deuxième mais cette fois ci ajoute le facteur de cas confirmés. Nous remarquons donc que chaque pays doit faire autant de test en fonction de sa population. Par exemple comparons les États-Unis et le Canada, pour confirmer 9415 cas, les États unis font 124 milles tests et le canada 268 milles qui est presque le double pourtant la population est États-Unis est plus élevée que celle du Canada.

✱ Le quatrième graphique rassemble nos données sur les tests avec les données sur les cas confirmés. Le tableau répond à la question: combien de tests un pays a-t-il effectués pour trouver un cas de COVID-19? En Amérique du Sud, certains pays par exemple Uruguay et le Paraguay ont effectué un grand nombre de tests pour chaque cas confirmé. Pourtant en Ecuador et au Brésil effectue un nombre faible de test pour trouver un cas confirmé.
